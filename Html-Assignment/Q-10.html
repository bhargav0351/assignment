How can we club two or more rows or columns into a single row or column in an HTMLtable? With Example.

    • we can club two or more rows or columns into a single row or column in an html table using rowspan and colspan attributes.
    • 1.rowspan
    • The rowspan attribute specifies the number of rows that a cell should span.
    • 2.colspan
    • The colspan attribute specifies the number of columns that a cell should span.
