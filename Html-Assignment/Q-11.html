What is the difference between a block-level element and an inline element?

    • 1.block element
    • Block level element is start with second line. or start with new line and it is obtain the full width of parent element.
    • block level element have top and bottom margin.
    • 2.inline element
    • Inline elements do not start on a new line and only take up as much width as necessary.
    • inline level element dose ont have top and bottom margin.
