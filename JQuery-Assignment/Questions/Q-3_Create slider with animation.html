Q :- 3 How to create slider with animation?

  1. jQuery offers various built-in effects for animations:
    • $(selector).hide(speed, callback); // Hide elements
    • $(selector).show(speed, callback); // Hide elements
    • $(selector).toggle(speed, callback); // Hide elements
    • $(selector).fadeIn(speed, callback); // Hide elements
    • $(selector).fadeOut(speed, callback); // Hide elements
    • $(selector).fadeToggle(speed, callback); // Hide elements
    • $(selector).slideUp(speed, callback); // Hide elements
    • $(selector).slideDown(speed, callback); // Hide elements
    • $(selector).slideToggle(speed, callback); // Hide elements
